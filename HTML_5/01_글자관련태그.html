글자 관련 태그

b 태그 : 단순히 글자를 굵게 표시하는 태그

i태그 : 단순히 글자를 기울이는 태그 ABC(태그는 중복이 가능하다)
strong태그 : 중요성, 심각성 등을 표시하는 강조 태그는
em태그 : 문장 일부의 중요성을 강조할 때 사용

mark태그 : 형광펜 효과를 나타내는 태그
u태드 : under Line(밑줄)을 나타내는 태그
s 태그 : 취소선(잘못됨을 표시)
del태그 : 삭제선
small태그 : 작은 글자 표시
sup태그(윗첨자) : x2
sub 태그(아랫첨자) : log210
