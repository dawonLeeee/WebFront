label 태그

        - input 태그의 제목으로 사용되는 태그를

        - 사용방법 : 
            (1) input태그에 id속성 작성
            (2) 원하는 위치에 label 태그를 사용한 후 for 속성을 작성
            이 때, for 속성의 값은 label태그의 속성 값과 같아야 한다

form태그

        - 내부에 작성된 input태그의 값을 서버로 전달
        - 이 때 전달되는 값의 형식은 k:v 형태

        key == input태그의 name 속성 값
        value == input태그에 작성된 값 또는 value속성값 

        - form 태그의 속성
            1) action : 내부에 작성된 input값을 어디로 전달할지 지정하는 속성
                ->주소를 작성하게 됨
            2) method(방식) : 데이터를 서버로 전달할 대의 방식을 지정하는 속성
            3) name : form 태그의 이름을 지정(JS 사용)
            4) target : action에 지정된 주소를 새 탭/현재 탭 중 어디서 열지 결정
